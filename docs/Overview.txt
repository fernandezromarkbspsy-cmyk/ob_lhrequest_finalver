Overview
A modern, sleek web application to manage Linehaul truck requests for Shopee Sorting Facility Dispatch department, replacing the current dual Google Sheets workflow with a streamlined approval system featuring sidebar navigation and real-time notifications.
User Authentication (Sheet-based)
Login Flow

App opens → Animated login popup modal appears login form
Role selection:

FTE (for FTE Ops / FTE MM users)
Backroom (for Ops PIC users)


User stored in database

User Identification Logic
ConditionRole IdentifiedOps id is NULL AND email is NOT NULLFTE (FTE Ops or FTE MM)Ops id is NOT NULL AND email is NULLOps PIC (Backroom)
Login Modal Features

Smooth fade-in animation after 5-second delay
Glassmorphism/blur backdrop effect
Two login tabs: FTE and Backroom
FTE Login: Email input → validates against user sheet
Backroom Login: Ops ID input → validates against user sheet
Error shake animation for invalid credentials
Success animation before redirect


User Roles

Ops PIC (Backroom) - Creates linehaul requests
FTE Ops (FTE) - Approves/Rejects/Edits requests (with sound alert)
FTE MM (FTE) - Confirms with Plate # or Rejects with Remarks (with sound alert)

Data Flow
Ops PIC (Create) → FTE Ops (Approve/Reject/Edit) → FTE MM (Confirm Plate # / Reject with Remarks)
